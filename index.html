Scales

A scale is a function that takes an input of data (of a defined domain) and returns an output of data (of a defined range.) In practice, this is used to normalize data for visualization purposes.

Use 1

Scales can be used to keep a data visualization program operational even as the domain of the data changes. By using the min and max functions, the scale function can create an array of output data that is normalized. This prevents the program from breaking if new data is outside the bounds of the original min and max.

Use 2

Scales can also be used to directly impact the visualization itself. The dimensions of visual elements can be tied to the output of the scale functions. For example, the radius of a data point's circle can be proportional to its output from the scale function.
